### 命令行

| 命令      | 用途         |
|:----------|:-------------|
| pwd       | 显示当前路径 |
| clear     | 清屏         |
| ls        | 列出目录内容 |
| cd        | 改变当前路径 |
| mkdir     | 创建文件夹   |
| touch     | 创建空文件   |
| cp        | 文件复制     |
| rm        | 文件删除     |
| mv        | 改名或移动   |
| date      | 显示如期     |
| echo      | 显示值       |
